## Kurulum {#installation}

Kolayca yükleyebilir ve dclist.net nodejs sdk ini kullanmaya başlayabilirsiniz. Sdk i [npm](https://www.npmjs.com/) ve [yarn](https://classic.yarnpkg.com) üzerinden indirebilirsiniz.

Yüklemek için aşağıdaki komutları kopyalayıp yapıştırabilirsiniz :


```bash
npm i —save @dclist /sdk
```

VEYA

```bash
yarn add @dclist/sdk
```

## Typescript Desteği {#typescript-support}

Typescript i destekliyoruz. Tüm kodlarımız strongly-typed kodlardır. Sdk in kendiside typescript kullanılarak yazılmıştır.

Paket açık kaynak kodludur. İstersen kodlarına göz atabilirsin. İşte [GitHub Sayfası](https://github.com/dclist/node-sdk).
