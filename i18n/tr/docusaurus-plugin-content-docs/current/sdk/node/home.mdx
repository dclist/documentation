## Installation {#installation}

You can easily install and start using dclist.net nodejs sdk. Sdk is available on [npm](https://www.npmjs.com/) and [yarn](https://classic.yarnpkg.com).

You copy-paste following commands to install :


```bash
npm i --save @dclist/sdk
```

OR

```bash
yarn add @dclist/sdk
```

## Typescript Support

We support typescript. All of our code is strongly-typed. Sdk itself is written in typescript.

This package is open sourced. You can take a look at its code if you want. Here is its [GitHub Page](https://github.com/dclist/node-sdk).
